# Onboarding-Tutorial

Das interaktive Tutorial führt neue Nutzer in 7 Schritten durch einen echten Beispielauftrag und erklärt die wichtigsten Funktionen.

## Automatischer Start

Das Tutorial startet automatisch beim ersten Login. Alle Daten im Tutorial sind temporär und werden **nicht in der Datenbank gespeichert**.

## Tutorial erneut starten

1. Klicke in der linken Sidebar ganz unten auf **Tutorial starten** (Hut-Symbol)
2. Das Tutorial beginnt von vorne auf der Auftragsseite

## Die 7 Schritte

| Schritt | Funktion |
|---------|----------|
| 1 | Kanban-Board — Auftrag per Drag & Drop verschieben |
| 2 | Auftragsdetail öffnen |
| 3 | 3D-Modell prüfen und kommentieren |
| 4 | Angebotsfreigabe an Kundin schicken |
| 5 | Druckjobs automatisch planen |
| 6 | Druckergebnis prüfen und freigeben |
| 7 | Abschluss |

## Tutorial überspringen

Im Willkommen-Dialog oder über das **×** in der Schritt-Einblendung.
