# Planning

The planning area provides a calendar-based overview of upcoming [[Print Jobs]] and [[Settings|machine]] utilization.

## Calendar view

Shows planned and running jobs on a timeline. Click a job to open it.

## Resource view

Groups jobs by machine and shows which machine is occupied on which day — useful when scheduling new jobs.

## Tips

- Create new jobs directly under [[Print Jobs]].
- The planning view is read-only — edit jobs by opening them.
