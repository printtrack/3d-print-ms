# Onboarding Tutorial

The interactive tutorial walks new users through a real sample order in 7 steps, explaining the key features.

## Auto-start

The tutorial starts automatically on first login. All tutorial data is temporary and **not saved to the database**.

## Restart the tutorial

1. Click **Start tutorial** (mortarboard icon) at the bottom of the left sidebar
2. The tutorial restarts on the orders page

## The 7 steps

| Step | Feature |
|------|---------|
| 1 | Kanban board — drag & drop an order to a new phase |
| 2 | Open order details |
| 3 | Inspect and annotate the 3D model |
| 4 | Send a price quote for customer approval |
| 5 | Auto-plan print jobs |
| 6 | Verify the print result |
| 7 | Completion |

## Skipping the tutorial

Click **Skip** in the welcome dialog or the **×** on any step tooltip.
